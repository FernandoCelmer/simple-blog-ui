## Overriding tokens

Override any token on `:root` to re-theme the library globally:

```css
:root {
  --primary: #1e40af;
  --color-red: #b91c1c;
}
```

Or scope overrides to a subtree by setting them on a parent element:

```css
.compact {
  --bg-subtle: #e5e5e5;
}
```

## Example usage

```css
.my-element {
  color: var(--text);
  background-color: var(--background);
  border: 2px solid var(--primary);
}

.my-alert--error {
  border-color: var(--color-red);
  color: var(--color-red);
}
```
