# Stack

Flexbox layout primitive. Props: `direction` (row/column), `align`, `justify`, `gap` (1–6), `wrap`.

## Usage

<FrameworkTabs
  html={`<sb-stack direction="row" gap="3" align="center">
  <sb-button>One</sb-button>
  <sb-button>Two</sb-button>
</sb-stack>`}
  react={`import { Stack, Button } from 'simple-blog-ui/react';

<Stack direction="row" gap="3" align="center">
  <Button>One</Button>
  <Button>Two</Button>
</Stack>`}
  vue={`<sb-stack direction="row" gap="3" align="center">
  <sb-button>One</sb-button>
  <sb-button>Two</sb-button>
</sb-stack>`}
/>
