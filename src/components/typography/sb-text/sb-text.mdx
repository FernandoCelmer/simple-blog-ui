# Text

Paragraph block with four variants (`body`, `muted`, `small`, `italic`) and optional bold weight.

<Text variant="body">The quick brown fox.</Text>

  vue={`<sb-text variant="body">The quick brown fox.</sb-text>
<sb-text variant="muted">Secondary info.</sb-text>`}
/>
