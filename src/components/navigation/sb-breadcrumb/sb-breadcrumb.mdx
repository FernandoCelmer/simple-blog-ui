# Breadcrumb

Trail of links showing the current path. Items are `{ label, href? }` — the last item is rendered as current page (not a link).

## Usage

<FrameworkTabs
  html={`<sb-breadcrumb id="crumb"></sb-breadcrumb>

<script>
  document.getElementById('crumb').items = [
    { label: 'Home', href: '/' },
    { label: 'Blog', href: '/blog' },
    { label: 'Post title' },
  ];
</script>`}
  react={`import { Breadcrumb } from 'simple-blog-ui/react';

const items = [
  { label: 'Home', href: '/' },
  { label: 'Blog', href: '/blog' },
  { label: 'Post title' },
];

<Breadcrumb items={items} separator="›" />`}
  vue={`<script setup>
const items = [
  { label: 'Home', href: '/' },
  { label: 'Blog', href: '/blog' },
  { label: 'Post title' },
];
</script>

<template>
  <sb-breadcrumb :items.prop="items" separator="›" />
</template>`}
/>
