# Link

Styled anchor. Three variants: `text` (default, underlined inline), `nav` (uppercase menu item), `muted` (gray).

## Usage

<FrameworkTabs
  html={`<sb-link href="/about">About</sb-link>
<sb-link variant="nav" href="/posts">Posts</sb-link>
<sb-link variant="muted" href="/archive">archive</sb-link>`}
  react={`import { Link } from 'simple-blog-ui/react';

<Link href="/about">About</Link>
<Link variant="nav" href="/posts">Posts</Link>
<Link variant="muted" href="/archive">archive</Link>`}
  vue={`<sb-link href="/about">About</sb-link>
<sb-link variant="nav" href="/posts">Posts</sb-link>
<sb-link variant="muted" href="/archive">archive</sb-link>`}
/>
