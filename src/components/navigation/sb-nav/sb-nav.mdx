# Nav

Navigation list rendered from an `items` array of `{ id, label, href }`. Orientation `horizontal` or `vertical`. Variants `solid` (filled active) or `underline`.

## Usage

<FrameworkTabs
  html={`<sb-nav id="nav" active="posts"></sb-nav>

<script>
  document.getElementById('nav').items = [
    { id: 'home', label: 'Home', href: '/' },
    { id: 'posts', label: 'Posts', href: '/posts' },
    { id: 'about', label: 'About', href: '/about' },
  ];
</script>`}
  react={`import { Nav } from 'simple-blog-ui/react';

const items = [
  { id: 'home', label: 'Home', href: '/' },
  { id: 'posts', label: 'Posts', href: '/posts' },
  { id: 'about', label: 'About', href: '/about' },
];

<Nav items={items} active="posts" />`}
  vue={`<script setup>
const items = [
  { id: 'home', label: 'Home', href: '/' },
  { id: 'posts', label: 'Posts', href: '/posts' },
  { id: 'about', label: 'About', href: '/about' },
];
</script>

<template>
  <sb-nav :items.prop="items" active="posts" />
</template>`}
/>
