# Table

Data-driven table with 2px solid borders matching the blog aesthetic. Pass `columns` (`[{ id, label, align? }]`) and `rows` (array of objects keyed by column id). Modifiers: `striped`, `compact`.

## Usage

<FrameworkTabs
  html={`<sb-table id="t" striped></sb-table>

<script>
  const t = document.getElementById('t');
  t.columns = [
    { id: 'cmd', label: 'Command' },
    { id: 'desc', label: 'Description' },
  ];
  t.rows = [
    { cmd: 'mkdocs serve', desc: 'Start server.' },
    { cmd: 'mkdocs build', desc: 'Build site.' },
  ];
</script>`}
  react={`import { Table } from 'simple-blog-ui/react';

const columns = [
  { id: 'cmd', label: 'Command' },
  { id: 'desc', label: 'Description' },
];

const rows = [
  { cmd: 'mkdocs serve', desc: 'Start server.' },
  { cmd: 'mkdocs build', desc: 'Build site.' },
];

<Table columns={columns} rows={rows} striped />`}
  vue={`<script setup>
const columns = [
  { id: 'cmd', label: 'Command' },
  { id: 'desc', label: 'Description' },
];
const rows = [
  { cmd: 'mkdocs serve', desc: 'Start server.' },
  { cmd: 'mkdocs build', desc: 'Build site.' },
];
</script>

<template>
  <sb-table :columns.prop="columns" :rows.prop="rows" striped />
</template>`}
/>
