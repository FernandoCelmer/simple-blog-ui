# Card

Bordered container with three slots: `header`, default (body), `footer`. Padding sizes `sm`, `md`, `lg`.

## Usage

<FrameworkTabs
  html={`<sb-card padding="md">
  <span slot="header">Article</span>
  Body content goes here.
  <sb-button slot="footer" size="sm">Read more</sb-button>
</sb-card>`}
  react={`import { Card, Button } from 'simple-blog-ui/react';

<Card padding="md">
  <span slot="header">Article</span>
  Body content goes here.
  <Button slot="footer" size="sm">Read more</Button>
</Card>`}
  vue={`<sb-card padding="md">
  <span slot="header">Article</span>
  Body content goes here.
  <sb-button slot="footer" size="sm">Read more</sb-button>
</sb-card>`}
/>
