# Accordion

Collapsible sections. Pass `items` as an array of `{ id, title }` and provide panel content via named slots. Set `multiple` to allow multiple panels open at once.

## Usage

<FrameworkTabs
  html={`<sb-accordion id="acc">
  <div slot="what">A UI library for simple-blog.</div>
  <div slot="how">npm install simple-blog-ui</div>
</sb-accordion>

<script>
  document.getElementById('acc').items = [
    { id: 'what', title: 'What is it?' },
    { id: 'how', title: 'How do I install?' },
  ];
</script>`}
  react={`import { Accordion } from 'simple-blog-ui/react';

const items = [
  { id: 'what', title: 'What is it?' },
  { id: 'how', title: 'How do I install?' },
];

<Accordion items={items}>
  <div slot="what">A UI library for simple-blog.</div>
  <div slot="how">npm install simple-blog-ui</div>
</Accordion>`}
  vue={`<script setup>
const items = [
  { id: 'what', title: 'What is it?' },
  { id: 'how', title: 'How do I install?' },
];
</script>

<template>
  <sb-accordion :items.prop="items">
    <div slot="what">A UI library for simple-blog.</div>
    <div slot="how">npm install simple-blog-ui</div>
  </sb-accordion>
</template>`}
/>
