# Collapse

Inline toggle that reveals hidden content via a trigger button. Emits `sb-toggle` on each open/close.

## Usage

<FrameworkTabs
  html={`<sb-collapse label="Show details">
  Additional info appears when expanded.
</sb-collapse>`}
  react={`import { Collapse } from 'simple-blog-ui/react';

<Collapse label="Show details" onToggle={(e) => console.log(e.detail.open)}>
  Additional info appears when expanded.
</Collapse>`}
  vue={`<sb-collapse label="Show details" @sb-toggle="onToggle">
  Additional info appears when expanded.
</sb-collapse>`}
/>
