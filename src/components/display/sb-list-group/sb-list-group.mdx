# List Group

Vertical list with divided items. Items are `{ id, label, meta?, disabled? }`. Enable `interactive` to make items clickable (emits `sb-select`).

## Usage

<FrameworkTabs
  html={`<sb-list-group id="list" interactive></sb-list-group>

<script>
  const list = document.getElementById('list');
  list.items = [
    { id: 'a', label: 'First', meta: '2d ago' },
    { id: 'b', label: 'Second', meta: '5d ago' },
  ];
  list.addEventListener('sb-select', (e) => console.log(e.detail.id));
</script>`}
  react={`import { ListGroup } from 'simple-blog-ui/react';

const items = [
  { id: 'a', label: 'First', meta: '2d ago' },
  { id: 'b', label: 'Second', meta: '5d ago' },
];

<ListGroup items={items} interactive onSelect={(e) => console.log(e.detail.id)} />`}
  vue={`<script setup>
const items = [
  { id: 'a', label: 'First', meta: '2d ago' },
  { id: 'b', label: 'Second', meta: '5d ago' },
];
</script>

<template>
  <sb-list-group :items.prop="items" interactive @sb-select="onSelect" />
</template>`}
/>
