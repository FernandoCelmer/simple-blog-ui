# Avatar

Square avatar with image (`src`) or initials fallback. Sizes `sm`, `md`, `lg`.

## Usage

<FrameworkTabs
  html={`<sb-avatar initials="FC" size="md"></sb-avatar>
<sb-avatar src="https://github.com/FernandoCelmer.png" alt="Fernando" size="lg"></sb-avatar>`}
  react={`import { Avatar } from 'simple-blog-ui/react';

<Avatar initials="FC" size="md" />
<Avatar src="https://github.com/FernandoCelmer.png" alt="Fernando" size="lg" />`}
  vue={`<sb-avatar initials="FC" size="md"></sb-avatar>
<sb-avatar src="https://github.com/FernandoCelmer.png" alt="Fernando" size="lg"></sb-avatar>`}
/>
