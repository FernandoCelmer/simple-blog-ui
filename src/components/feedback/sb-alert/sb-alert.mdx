# Alert

Inline message block with semantic variants (`info`, `success`, `warning`, `danger`). Add `dismissible` for a × that emits `sb-dismiss`.

## Usage

<FrameworkTabs
  html={`<sb-alert variant="success" title="Saved">
  Your changes were saved.
</sb-alert>

<sb-alert variant="danger" title="Error" dismissible>
  Something went wrong.
</sb-alert>`}
  react={`import { Alert } from 'simple-blog-ui/react';

<Alert variant="success" title="Saved">
  Your changes were saved.
</Alert>

<Alert variant="danger" title="Error" dismissible onDismiss={() => {}}>
  Something went wrong.
</Alert>`}
  vue={`<sb-alert variant="success" title="Saved">
  Your changes were saved.
</sb-alert>

<sb-alert variant="danger" title="Error" dismissible @sb-dismiss="onDismiss">
  Something went wrong.
</sb-alert>`}
/>
