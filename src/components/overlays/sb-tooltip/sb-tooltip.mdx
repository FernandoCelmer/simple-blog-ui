# Tooltip

Short hint shown on hover or focus. `placement` can be `top` (default), `bottom`, `left`, or `right`. Wraps any element.

## Usage

<FrameworkTabs
  html={`<sb-tooltip label="Keyboard shortcut: K" placement="top">
  <sb-button>Hover me</sb-button>
</sb-tooltip>`}
  react={`import { Tooltip, Button } from 'simple-blog-ui/react';

<Tooltip label="Keyboard shortcut: K" placement="top">
  <Button>Hover me</Button>
</Tooltip>`}
  vue={`<sb-tooltip label="Keyboard shortcut: K" placement="top">
  <sb-button>Hover me</sb-button>
</sb-tooltip>`}
/>
