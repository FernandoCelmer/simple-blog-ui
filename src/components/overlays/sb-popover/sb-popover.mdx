# Popover

Richer tooltip with optional title and multi-line content. `trigger` can be `click` (default) or `hover`. Supports same four placements as tooltip.

## Usage

<FrameworkTabs
  html={`<sb-popover title="What's new" placement="bottom" trigger="click">
  <sb-button slot="trigger">Changelog</sb-button>
  Version 0.2.0 adds dark mode, toast, and switch.
</sb-popover>`}
  react={`import { Popover, Button } from 'simple-blog-ui/react';

<Popover title="What's new" placement="bottom" trigger="click">
  <Button slot="trigger">Changelog</Button>
  Version 0.2.0 adds dark mode, toast, and switch.
</Popover>`}
  vue={`<sb-popover title="What's new" placement="bottom" trigger="click">
  <sb-button slot="trigger">Changelog</sb-button>
  Version 0.2.0 adds dark mode, toast, and switch.
</sb-popover>`}
/>
