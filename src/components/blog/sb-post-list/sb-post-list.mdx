# PostList

List of blog posts with date + title. Each item has `date`, `title`, `url`. Stacks on mobile.

## Usage

<FrameworkTabs
  html={`<sb-post-list></sb-post-list>
<script>
  document.querySelector('sb-post-list').posts = [
    { date: '2026-04-17', title: 'Post Title', url: '/posts/one' }
  ];
</script>`}
  react={`import { PostList } from 'simple-blog-ui/react';

<PostList posts={[
  { date: '2026-04-17', title: 'Post Title', url: '/posts/one' }
]} />`}
  vue={`<sb-post-list :posts="posts"></sb-post-list>`}
/>
