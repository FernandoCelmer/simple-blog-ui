# Sidebar

Nested sidebar navigation for docs/TOC. Accepts an `items` array where each item can have `title`, `url`, and optional `children`.

## Usage

<FrameworkTabs
  html={`<sb-sidebar></sb-sidebar>
<script>
  document.querySelector('sb-sidebar').items = [
    { title: 'Intro', url: '#intro' },
    { title: 'Guide', url: '#guide', children: [
      { title: 'Install', url: '#install' }
    ]}
  ];
</script>`}
  react={`import { Sidebar } from 'simple-blog-ui/react';

<Sidebar items={[
  { title: 'Intro', url: '#intro' },
  { title: 'Guide', url: '#guide', children: [
    { title: 'Install', url: '#install' }
  ]}
]} active="#install" />`}
  vue={`<sb-sidebar :items="items" active="#install"></sb-sidebar>`}
/>
