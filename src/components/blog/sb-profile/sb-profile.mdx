# Profile

Centered about/profile page with name, role, optional bio (default slot), and social links array.

## Usage

<FrameworkTabs
  html={`<sb-profile name="John Doe" role="Developer"></sb-profile>
<script>
  document.querySelector('sb-profile').links = [
    { label: 'GitHub', url: 'https://github.com/...' }
  ];
</script>`}
  react={`import { Profile } from 'simple-blog-ui/react';

<Profile name="John Doe" role="Developer" links={[
  { label: 'GitHub', url: '#' }
]} />`}
  vue={`<sb-profile name="John Doe" role="Developer" :links="links"></sb-profile>`}
/>
