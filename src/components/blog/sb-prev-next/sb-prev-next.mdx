# PrevNext

Three-column navigation for previous/next post links. Responsive — stacks on mobile.

## Usage

<FrameworkTabs
  html={`<sb-prev-next
  prev-href="/posts/intro"
  prev-label="Introduction"
  next-href="/posts/advanced"
  next-label="Advanced"
></sb-prev-next>`}
  react={`import { PrevNext } from 'simple-blog-ui/react';

<PrevNext
  prevHref="/posts/intro"
  prevLabel="Introduction"
  nextHref="/posts/advanced"
  nextLabel="Advanced"
/>`}
  vue={`<sb-prev-next
  prev-href="/posts/intro"
  prev-label="Introduction"
  next-href="/posts/advanced"
  next-label="Advanced"
></sb-prev-next>`}
/>
