# Textarea

Multi-line text input with label, hint, error. Same API as `sb-input` plus a `rows` prop. Emits `sb-input` on every keystroke.

## Usage

<FrameworkTabs
  html={`<sb-textarea
  label="Comments"
  placeholder="Tell us what you think..."
  rows="5"
></sb-textarea>`}
  react={`import { Textarea } from 'simple-blog-ui/react';

<Textarea
  label="Comments"
  placeholder="Tell us what you think..."
  rows={5}
  onInput={(e) => console.log(e.detail.value)}
/>`}
  vue={`<sb-textarea
  label="Comments"
  placeholder="Tell us what you think..."
  :rows="5"
  @sb-input="onInput"
/>`}
/>
