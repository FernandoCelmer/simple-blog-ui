# Checkbox

Square black-bordered checkbox with slotted label. Emits `sb-change` with `{ checked, value }`.

## Usage

<FrameworkTabs
  html={`<sb-checkbox name="newsletter">Subscribe to newsletter</sb-checkbox>

<script>
  document.querySelector('sb-checkbox')
    .addEventListener('sb-change', (e) => console.log(e.detail.checked));
</script>`}
  react={`import { Checkbox } from 'simple-blog-ui/react';

<Checkbox
  name="newsletter"
  onChange={(e) => console.log(e.detail.checked)}
>
  Subscribe to newsletter
</Checkbox>`}
  vue={`<sb-checkbox
  name="newsletter"
  @sb-change="onChange"
>
  Subscribe to newsletter
</sb-checkbox>`}
/>
